# 面试问题

  • 为什么从上家公司离职？
  • 为什么yarn比npm快？它们的区别是什么？（这道题跪了，我居然真的不知道。看来平时没有真正做到“知其然，并知其所以然”）
  • 你对CI/CD的理解是什么？
  • 你做过权限的控制吗？如何做？（回来才想到面试官想问的是路由守卫做页面的权限访问控制，但我答跑题了，不知道是不是我想的这样。这也使我意识到一个问题，以后在面试别人的时候，如果对方答跑题了，要么给他一些提示，要么这题不扣分）
  • 你在上一家公司是什么岗位，扮演什么角色？（岗位？当然是前端工程师啊，其实他想问的是我的职级，答：Web前端负责人）
  • 你们前端团队最多的时候有多少人？
  • 你认为前端管理最难的是什么？（人才的招聘，很难找到一个能在上面持续投入的人，而不是因为前端入门门槛低，找份工作而已）
  • 你们开发的一个流程是什么？前后端如何联调，代码又是如何部署的呢？
  • Vue和React的区别是什么？他们的应用场景是什么？
  • 你们用了Vue1.x 和 Vue2.x如何做版本的迁移，如果我想在同一个项目中使用两个版本的Vue该如何实现？（想了几秒：Webpack的alias）
  • 你们是如何使用Git的？如何管理版本？如何解决冲突？（这里答得让面试官不是很满意，他提出了git flow的东西，确实不知道这个）
  • 你在工作中遇到的最大困难是什么，是如何解决的？
  • 你在上家公司的薪资是多少？其他的津贴是多少？公积金按多少交？（薪资我实说了，公积金交得太低，不好意思说，于是编了个10%，对方说：“这么惨？”，我苦笑答：“是啊？”，但没问他们是多少，肯定比10%这还多。回来的路上在想，不应该怕丢脸而去说谎，再想想，应该是公司丢脸，而不是我丢脸，唉，当时我怎么会这么糊涂了。有些公司在你入职前，会要求你提供上一份工作的工资条或银行流水，所以薪资待遇这块，以后还是实话实说的好。）
  • 现在住哪？买房了吗？结婚了吗？有小孩吗？

# 我的问题

  • 上班时间（弹性工作制，不打卡）
  • 如果我进来的话，负责哪一部分的工作？
  • 贵公司的企业愿景是什么？（这么装逼的问题，我终于问出口了，他答了几句，我一句没记住 😦 ）

# 面试总结

路程时间：大概1小时10分，117元打车费用。而且太偏了，提前到了，本想找个地方坐下来再刷几道题，但周围全是厂房，只好提前进去，面试也提前了大概1小时。

面试整个过程大概1个小时左右，一些问题答得不是很好。有65%预感会过。

关于地址太遥远的问题，面试官说8月份会搬到市区，这点倒是蛮期待的。

最后还让我在他们公司吃了午饭才回来。回到家已下午3点过了。累～

等结果出来再做下一步的打算吧。

# 面试预告

去面试的路上，刚下出租车没多久，有个电话进来了，说是百度HR。我是投了一家本地的百度控股的公司，我以为没戏了。约我面试，我说只有等清明节后了。所以清明节后有个“保底”的了。
