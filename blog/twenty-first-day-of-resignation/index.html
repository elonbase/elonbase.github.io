# 番外

原来以为今天没有面试。早上起来将前几天的博客补完，中午就出去照相，用于下周的入职。中午拒绝了航天云网的Offer，而心里一直惦记着云景文旅的二面，心想，二面估计要安排到下周了，而下周要入职百讯智汇，怕到时候这边二面通过而且待遇也不错，如果选择云景文旅的话，那么可能要走那边的离职流程，有点麻烦。

所以，我给百讯智汇的HR发了微信，说能否延长3天入职，而且把我在等另一个公司的二面通知，想在两家公司做下选择。然而却不能如愿，她说她们不能接受延长时间入职，她们很缺人手，而且名额有限，如果我不按Offer上的时间入职的话，Offer就失效，把Offer给其他人，叫我考虑一下，18点下班前给她回复，绝情的语气（感受到了）。心中虽然有些不悦，但是也能理解。

于是想趁今天有时间，把之前承诺同事们的饭给请了。来到高新区公司附近，找了一家餐馆，准备就在那等同事们下班，然后一起吃饭聊聊天。坐下不久，云景文旅HR发来信息说领导那边一直在开会，不确定今天能安排二面，想安排到下周一，问我能不能将下周的入职时间往后推两天。

我把我跟百讯智汇HR的聊天截图发给了她，最后打了个笑哭的表情。然后她说，让我先去入职，到时候抽时间来二面就可以了。我于是就回复了百讯智汇的HR，说下周入职。她回复：好的哟 + 3个可爱表情。

好吧，那就这样决定了吧。我可以安心的和小伙伴们吃饭了。

没过多久，云景文旅HR发来信息说，现在是否有时间，可以过去进行二面。于是跟小伙伴们说了情况，说下次找时间再聚，于是充忙地拦了辆出租车就过去了。

# 面试问题

  • 自我介绍
  • 你对职业的规划是什么？
  • 你期望的薪资是多少？
  • 你之前是前端小组长，到这边可能是个普通的开发人员，你的心态会有什么变化吗？（我其实不想当组长啦）
  • ...

# 面试总结

二面的领导之前也是做开发的，穿着打扮还是一个程序员大装扮，头发有些油腻，发量貌似已不多，能透过刘海看见稍微发亮的前额头皮。领导没问技术问题，就跟我聊了公司的薪酬组成和公司的发展规划，公司服务的客户等。感觉领导还是挺随和的。整个过程大概十多分钟。

我说我是不是先回去等消息时，他叫我等一下，叫了HR主管跟我聊了一下，主要就是薪酬这块，她问我期望的薪资是多少时，我说了一个数，她说：这个应该没问题，这是我们公司的平均水平。我当时差点一口血喷在她脸上了。唉，我too young, too simple了。她们需要评估一下最终结果，然后走审批流程，然后才能给我最终结果。

我希望她们能在周末告诉我结果，这样我就能在入职前做出决定。
