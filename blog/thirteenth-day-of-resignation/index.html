# 笔试题

共有50道题，有单选、多选、填空、判断和简答题。时间给得很充裕，最多1个半小时。都是一些CSS，Vue，Sass的基础题，很奇怪，JavaScript的很少，甚至没有，太多记不清了。最后一道简答题（15分）是：如何实现v-model双向绑定？

# 面试题
  • 请介绍以下你之前的项目以及负责的部分？
  • 你们公司主要是做哪些项目的？
  • 你离职的原因真的是你所说的——想到更有挑战的环境中工作，获得职业能力的提升？没有其他原因？（没有，的确是这个）

# 我的问题

本来想问的，面试时间很短，大概20分钟的样子，感觉这次又挂了，所以没问。

# 面试总结

这次面试的人，总共有十来个人，有面项目经理的、产品经理的、Web前端工程师的、UI设计师的。

早上10:30左右开始笔试，下午差不多14:00点才开始面试。

大家13:30就开始在他们的一个会议室里等了，差不多14:00才开始面试，我排到了16:20左右才到我，等了几乎3个小时。

走进会议室，示意我坐下，有6个人做在对面，这种场面还是5年前面试一个事业单位时遇到过。从左边数的第三个，一个中年男人开始叫介绍以下之前项目以及负责的部分。说完过后，他示意左边第一个，一个30岁左右的女性开始问问题。她只是翻着我的简历，问上面我做的项目，大多都是在确认我做过。我感觉像是在录口供一样。

她问完后，示意第二个人可以开始，第二个人说没有问题。到第三个人，就刚刚第一个问我问题的人，随后又问了几个开放性的问题。示意下一个（第四个）可以开始，下一个摇头。再下一个（第五个，感觉快有60岁的一个老头），也没有问题。最边上坐的感觉是个财务，也没有问题。

最后说，今天面试结束。我感觉整个过程很短暂，心想这次肯定又是挂了，有些灰溜溜的从会议室里走出来。

这个地方的住处不远，走着回去吧。一路上我在想我哪里出问题了。笔试题是不是犯了一些不可原谅的基础知识错误？面试时，面试官问我离职的真实动机是什么，难道他有其他的考虑吗？越想越没底。

回到家，我在Boss上问他结果，高兴的是，他对我是认可的。后来我们加了微信，再次询问时，他说他们对我是认可的，等明天汇报领导，再和我谈后续事宜。

这算是今天小小的收获吧！

疑虑一： 进入大楼前，要登记个人信息，还要填被访人的电话号码，我填的是昨天打电话约我今天的来面试，叫我老师的小姑娘的电话。填完后，打印出一张小纸条，上面有我的个人信息、被访人的信息和一个用于通过闸机的二维码。我发现纸条上显示这个被访人小姑娘是派遣员工。这还是同样来面试的一个人发现跟我说的。等后续跟他们谈的时候，我一定要把这个搞清楚，派遣制的话，我不干。

疑虑二： 感觉今天的笔试和面试都太简单、太轻松，心里还是没底。

唉，昨天的面试，说是“肯定进二面”的，今天也没收到消息。这些不讲诚信的人，不靠谱，从现在开始不再去在意了。Keep moving！
