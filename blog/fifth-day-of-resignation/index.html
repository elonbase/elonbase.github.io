# 面试岗位

朗玛视讯 - IPTV前端开发

# 推荐星级 ⭐️

# 面试问题

  • 自我介绍。
  • 为什么从上家公司离职？
  • 你对以后的职业规划是什么？（无障碍设计，用户体验设计，前端要有使命感什么的说一通）

（只记得这么几个，总的也没几个，1/3的时间都是他在讲 😃 ）

这个公司是朗玛信息一个业务部门独立出来的子公司，IPTV方向的。但是办公还是在一起办公。这家公司其实好几年前来面试过，那时我还是个二十多岁的小伙子啊。

我一进就让我填一个叫“职位申请表”的表单，有2页，我看了差点晕了。要填家庭成员、上一份工作的薪资、紧急联系人。。。。等等一些不相关的东西。我在那里停顿了几十秒钟，决定不填一些不相关的项。但是我填了期望薪资，我填了一个范围，最左的一端填的是该职位在Boss上薪资范围的最右端，意思是，我最低要的数是你们能给的最高数。就是这么霸气！

诺大的办公室，但四周被墙或是玻璃墙围着，必须要开灯才能办公，我是真受不了这么密闭而没有窗户的办公环境，我待久了估计会抓狂的。HR是个穿着白色裙子年轻MM，挺漂亮的，但是脸上没什么表情，叫我在一个房间里等一下。这个房间和场景很熟悉，我几乎确认就是多年前来面试的那个房间。我心想，这难道真是造化弄人吗？

一个大概四十多数中年男人拿着我的简历进来了，聊了一下，知道就是在Boss上和我联系的“技术总监”。聊了几分钟，都没有问我一些技术的问题，我就感觉这次又黄了，是我要高了，人家只是礼貌的来和你聊聊天，我索性也就放开了聊。最后我也了解到，他们其实想招一个技术（要钱）不是那么高的，能做页面就行。虽然黄了，但其实，我并不伤心，我说过我对这个职位不是很敢兴趣，过来只是看看我了解的信息是否有偏差，或许会有我意想不到的惊喜。很显然没有。

整个面试过程大概30分钟，轻松而和谐。

希望明天好运吧，虽然明天是愚人节，愿老天爷别给我开玩笑。

回来的时候在想，明天有面试，那么后天呢？后天就落空了，有些失落感。

但是在下午5点过，来了一个电话，说能不能约明天面试，我说明天没时间，于是推到了后天。巧的是，约面试公司的是朗玛信息，早上面试公司的母公司。我没想到他们会给我面试机会，还是今天去的老地方。

我想到了后天去面的时候必须要问的一个问题：你们这有没有靠窗的或能看到外面世界的工位啊，如果有，我要是进来的话，我可以坐那吗？😃

# 面试预告

今天开心的事是，这次面试前10几分钟，我接到了早上投的白山云科技HR的面试电话，这是一家我比较心动的公司，至少从开的薪资上是心动的 😃 。但是在贵安新区，离城区很远。之前一直没投是因为几年前带的一个实习生在里面，我害怕去面试，怕他坐在面试官的位置上，我会显得很尴尬和窘迫。后面想想其实也没什么，这说明他出息了，也就索性投了。
