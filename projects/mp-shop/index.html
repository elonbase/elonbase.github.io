便利店微信小程序（Dead）

2019/12/01 - 2020/05/16

# 初衷

为店铺周边社区内居民及店铺顾客提供线上购物服务，并提供免费配送服务。通过线上小程序购物平台突破实体店的空间瓶颈，将服务扩散到店铺周边更大的范围，从而实现营业额的增长。

# 技术栈

  • 原型 - 墨刀 (opens new window)
  • UI设计 - Sketch
  • 客户端 - 微信小程序（Native）
  • 后台管理(Front-End) - React.js + Ant Design
  • 后台服务端 - Nodejs + MySql
  • 部署 - Ubuntu + Nginx + Jenkins + Docker Compose

# 目前状态

弃用，已不再维护。
